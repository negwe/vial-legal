# Politique de confidentialité — Vial

**Dernière mise à jour : mars 2026**

## L'essentiel

Vial ne collecte aucune donnée personnelle. Nous n'avons pas de serveur, pas de base de données, pas de compte utilisateur. Nous n'avons jamais accès à vos données. Point.

## 1. Où sont vos données ?

Toutes vos données restent sur votre appareil, dans un conteneur chiffré par iOS.

Si vous activez la synchronisation iCloud, vos données sont stockées dans votre espace CloudKit privé, géré et chiffré par Apple. Seul vous pouvez y accéder — Vial n'a aucun moyen de lire, consulter ou extraire ces données.

Quand votre appareil est verrouillé, toutes les données locales sont chiffrées et inaccessibles (iOS Data Protection).

## 2. Ce que Vial ne fait pas

- Aucune donnée n'est transmise à un serveur
- - Aucune donnée n'est partagée avec des tiers
  - - Aucune publicité, aucun profilage, aucun data mining
    - - Aucun cookie, aucun tracker, aucun outil d'analytics
      - - Aucun SDK tiers de mesure d'audience
        - - Aucun compte utilisateur à créer
         
          - ## 3. Scan d'ordonnances
         
          - Lorsque vous scannez une ordonnance, l'image est traitée entièrement sur votre appareil via le framework Vision d'Apple. L'analyse du texte est effectuée par Apple Intelligence (Foundation Models), également sur l'appareil.
         
          - L'image de l'ordonnance n'est jamais envoyée sur internet.
         
          - ## 4. Apple Santé (HealthKit)
         
          - Si vous choisissez d'importer vos médicaments depuis Apple Santé :
         
          - - Vial accède uniquement aux médicaments annotés (lecture seule)
            - - Vial n'écrit jamais dans Apple Santé
              - - Vous contrôlez cet accès dans Réglages > Santé > Vial
                - - Vous pouvez révoquer l'accès à tout moment
                 
                  - ## 5. Notifications
                 
                  - Vial utilise les notifications locales pour vos rappels de prise. Elles sont programmées et délivrées entièrement par iOS, sans passer par un serveur externe.
                 
                  - ## 6. Vos données vous appartiennent
                 
                  - Puisque vos données ne quittent jamais votre appareil (ou votre CloudKit privé), vous en avez le contrôle total :
                 
                  - - **Consulter** : toutes vos données sont visibles dans l'app
                    - - **Modifier** : vous pouvez éditer vos médicaments et votre historique
                      - - **Exporter** : générez un rapport PDF depuis les Réglages
                        - - **Supprimer** : Réglages > Données > Réinitialiser efface tout
                         
                          - La suppression de l'app entraîne la suppression définitive de toutes les données locales.
                         
                          - ## 7. Modifications
                         
                          - Cette politique peut être mise à jour. La date de dernière modification est indiquée en haut de cette page.
